# 液压与气压传动 · 实验规范检测系统

## 功能概述

本系统为《液压与气压传动》课程开发的 **AI 辅助教学工具**，面向实验报告文本、实验台照片与操作视频，帮助教师快速发现结构缺失、数据不规范与潜在安全风险。

**重要说明：** 系统不替代教师批改与最终评分，所有结论需经教师确认。

## 课程信息

见项目根目录 `液压与气压传动.json`（授课教师：宋正河，2024级车辆工程，工学院）。

## 系统架构

```
前端 (frontend/)  ──HTTP──►  后端 FastAPI (backend/)
                                    │
                    ┌───────────────┴───────────────┐
                    ▼                               ▼
         本地 Qwen（OpenAI 兼容 API）          OpenCV 抽帧（仅视频）
         文本 / 图片 / 多帧 均由大模型分析
```

## 统一输出 JSON

| 字段 | 说明 |
|------|------|
| score | 0–100 总分 |
| level | pass / warning / fail |
| issues | 问题列表（含 category、severity、problem、evidence、suggestion、confidence） |
| teacher_review_required | 是否建议教师复核 |
| summary | 总体评价 |
| key_events | 视频专用时间段事件 |

## API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/hydraulic-lab/check-text` | 实验报告 JSON 体 |
| POST | `/api/hydraulic-lab/check-image` | multipart：image, experiment_type, grade |
| POST | `/api/hydraulic-lab/check-video` | multipart：video, experiment_type, frame_interval, grade |
| GET | `/api/health` | 健康检查与 AI 是否启用 |

## 启动方式

```bash
cd backend
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

浏览器打开 `frontend/index.html`（或将 frontend 用任意静态服务器托管），API 地址填 `http://127.0.0.1:8000`。

**所有检测均依赖大模型**，须先启动本机 Qwen（推荐 Ollama）。复制 `backend/.env.example` 为 `.env`，执行 `ollama pull qwen2.5:7b` 与 `ollama pull qwen2.5vl:7b` 后重启后端。模型不可用则接口返回 503。
