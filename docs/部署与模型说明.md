# 部署与模型说明

## 检测模式

本系统为 **`llm_only`**：文本、图片、视频（抽帧后）**全部**由大模型生成检测报告。

- 不再使用关键词规则引擎作为检测结果来源
- 大模型未启动或调用失败时，接口返回 **HTTP 503**，前端显示错误信息

## 必备服务

| 检测类型 | 模型能力 | 推荐模型（Ollama） |
|----------|----------|-------------------|
| 文本报告 | 指令模型 | `qwen2.5:7b` |
| 图片 | 视觉多模态 | `qwen2.5vl:7b` |
| 视频 | 视觉多模态（多帧） | `qwen2.5vl:7b` |

```powershell
ollama pull qwen2.5:7b
ollama pull qwen2.5vl:7b
ollama serve
```

## 配置

复制 `backend/.env.example` → `backend/.env`，按实验室服务器修改地址与模型名。

验收前访问：`GET http://127.0.0.1:8000/api/health`  
应看到 `"detection_mode": "llm_only"` 且 `"ai_enabled": true`。

## 交付说明

交付**代码 + Skill + 提示词 + 部署文档**，不交付模型权重。部署方在目标机器安装 Ollama/vLLM 并拉取 Qwen 模型即可。
